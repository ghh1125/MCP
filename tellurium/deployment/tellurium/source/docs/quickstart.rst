Quick Start
===========

To get started using Tellurium, either install Tellurium in your favorite Python environment with

::

    pip install tellurium

or install the latest `Tellurium/Spyder front end <https://sourceforge.net/projects/pytellurium/files/>`__.  For more details, see `the Tellurium README <https://github.com/sys-bio/tellurium?tab=readme-ov-file#installation-instructions>`__.
