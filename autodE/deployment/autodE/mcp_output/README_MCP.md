# autodE MCP (Model Context Protocol) Service README

## 1) Project Introduction

This service exposes key `autodE` capabilities through MCP (Model Context Protocol): automated molecular/reaction workflows, conformer generation, transition-state search, NEB path generation, and quantum-chemistry backend execution.

Primary goals:
- Build molecular/species objects from coordinates or SMILES
- Run single calculations via external electronic-structure engines
- Orchestrate reaction profiles and transition-state localization
